# AdventureWorks SQL Analysis

## Project Overview

This project uses the AdventureWorks 2022 database to perform business-oriented SQL analysis. The goal is to extract insights related to sales performance, products, customers, and revenue.

## Objectives

* Practice advanced SQL querying.
* Analyze sales trends and business performance.
* Generate KPIs using SQL.
* Demonstrate data analysis skills.

## Tools & Technologies

* SQL Server
* AdventureWorks 2022

## SQL Concepts Used

* SELECT Statements
* INNER JOIN
* LEFT JOIN
* Common Table Expressions (CTEs)
* Aggregate Functions
* GROUP BY
* Subqueries
* Window Functions

## Business Questions

* What are the best-selling products?
* Which customers generate the highest revenue?
* How have sales evolved over time?
* Which territories produce the highest sales volume?

## Key Findings

* Identified top-performing products.
* Analyzed customer purchasing behavior.
* Calculated revenue trends over time.
* Compared sales performance across regions.
